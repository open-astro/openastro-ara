OpenAstro Ara — Third-party License Inventory
==============================================

This file lists the third-party components bundled with OpenAstro Ara binary
distributions (server + Flutter client) along with their licenses. The list
covers runtime dependencies only; build-time tooling (compilers, test
frameworks, etc.) is not enumerated.

For the canonical lineage attribution + per-file copyright header
conventions, see NOTICE.md.

This file is curated for v0.0.1-ara.1. Future releases regenerate it from
the dotnet + pubspec package graphs.

===================================================================
